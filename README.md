# 🧮 Kalkulator Modern

Nje kalkulator i thjeshte dhe modern i ndertuar me HTML, CSS dhe JavaScript.

## ✨ Funksionaliteti
- Operacionet baze: mbledhje, zbritje, shumezim, pjesetim
- Shenja pozitive/negative dhe perqindje
- Tastiera e keyboards-it e mbeshtetura
- Dizajn responsive me dark theme

## 🚀 Si ta perdoresh
1. Hap `index.html` ne browser
2. Klikoni butonat ose perdorni tastieren

## 🛠️ Teknologjite
- HTML5
- CSS3 (Grid, Variables, Backdrop Filter)
- JavaScript (Vanilla)
